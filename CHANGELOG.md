# История изменений

## Версия 1.2.3, 2 июня 2026 г.

- Добавлена поддержка изменения размера терминала и центрированного отображения текста.
- Исправлена ошибка при отображении текста в терминале с неправильным размером.
- Переименована функция `getdate` в `dos_getdate` для совместимости с MacOS.

## Версия 1.2.2, 2 июня 2026 г.

- Переименованы массивы баннеров для улучшения читаемости и соответствия их назначению.
- Изменил цвет баннера "МОЛОДЕЦ" с `LIGHTGREEN` на `GREEN`
- Изменил цвет баннера "НЕВЕРНО" с `LIGHTGREEN` на `RED`

## Версия 1.2.1, 2 июня 2026 г.

- Исправлено восстановление терминала при аварийном завершении по `Ctrl+C`.
- Добавлена обработка сигналов `SIGINT`, `SIGTERM` и `SIGQUIT` для возврата
  из альтернативного экрана терминала, показа курсора и сброса цветов.
- При прерывании программы теперь восстанавливается raw-режим терминального
  ввода перед завершением процесса.
- В `.gitignore` добавлен каталог сборки `cmake-build-release`.

## Версия 1.2.0, 2 июня 2026 г.

- Проект портирован с Turbo C/DOS на GCC/Linux.
- Добавлена сборка через CMake.
- Исходные файлы и заголовки переименованы в нижний регистр для удобной
  сборки на Linux.
- Иконка и PIF-файл переименованы в нижний регистр: `arith.ico` и `arith.pif`.
- Удален DOS-исполняемый файл `ARITH.EXE` из репозитория.
- Удален старый проектный файл Turbo C `ARITH.PRJ`.
- Сгенерированный файл настроек `a.dat` добавлен в `.gitignore`.
- Добавлен слой совместимости `dos_compat` для функций Turbo C/DOS:
  `randomize`, `delay`, `sound`, `nosound`, `getdate`, `gettime`,
  `getftime` и связанных структур.
- Удалены Turbo C inline-assembler-блоки; функции BIOS/DOS/мыши заменены
  на переносимые реализации или временные заглушки.
- Реализован реальный терминальный вывод через ANSI-последовательности:
  позиционирование курсора, очистка экрана и стандартные цвета терминала.
- Добавлен ввод с терминала через `termios` в неблокирующем raw-режиме.
- Добавлена обработка ANSI/CSI-последовательностей для клавиш `Esc`,
  `Enter`, `Tab`, стрелок, `Home`, `End`, `PageUp`, `PageDown`, `F1`-`F8`
  и `F10`.
- Добавлена поддержка UTF-8 для русского текста, горячих клавиш, ввода имени
  и транслитерации имени в файл результата.
- Исправлена работа курсора и стирания при вводе UTF-8 имени: русские символы
  больше не считаются как два экранных знакоместа и не стираются по половине.
- Добавлено переключение в альтернативный экран терминала и восстановление
  состояния терминала после завершения программы.
- Пример файла настроек перенесен из `A.DAT` в `examples/a.dat`.
- Добавлен заголовок `arith.h` с объявлениями функций основного модуля.
- В интерфейсе обновлен номер версии до `1.2.0`.
- В стартовом экране подровнены заголовок и год выпуска.

## Версия 1.1.1, октябрь 1999 г. (old v1.02)

- Исправлена ошибка при выводе результатов.
- Исправлена ошибка при образовании имени файла результатов.

## Версия 1.1.0, август 1997 г. (old v1.01)

- Добавлено действие деления с остатком.
- В протокол результатов добавлен вывод лучшего, худшего и среднего времени.

## Версия 1.0.0, август 1997 г. (old v1.00)

- Действия: сложение, вычитание, умножение, деление.
- Текстовый режим.
- Сохранение протокола в текстовом файле по введенному имени и расширению
  `000`-`999`.
- Просмотр протокола последнего сеанса.
- Сохранение параметров в файле.
- Иконка и PIF-файл для Windows.
- Заглушка хранителя экрана Windows.
