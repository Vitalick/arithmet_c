# История изменений

## Версия 1.1.1, октябрь 1999 г. (old v1.02)

- Исправлена ошибка при выводе результатов.
- Исправлена ошибка при образовании имени файла результатов.

## Версия 1.1.0, август 1997 г. (old v1.01)

- Добавлено действие деления с остатком.
- В протокол результатов добавлен вывод лучшего, худшего и среднего времени.

## Версия 1.0.0, август 1997 г. (old v1.00)

- Действия: сложение, вычитание, умножение, деление.
- Текстовый режим.
- Сохранение протокола в текстовом файле по введенному имени и расширению
  `000`-`999`.
- Просмотр протокола последнего сеанса.
- Сохранение параметров в файле.
- Иконка и PIF-файл для Windows.
- Заглушка хранителя экрана Windows.
