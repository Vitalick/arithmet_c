# Устный счет

Программа **Устный счет** предназначена для контроля (проверки) умения
считать в уме.

Может быть использована как в школе, так и в домашних условиях.

Обеспечивает настройку, в которую включено ввод имени, выбор сложности,
и интервала чисел, а также выбор действий.

После выполнения решений можно посмотреть результат работы, который
сохраняется в файле.

## Требования к системе

Никаких требований нет!

Если программа запускается на Вашем компьютере, то она на нем работает.

А для запуска необходимо иметь операционную систему DOS или WINDOWS.
Вот и все!

## Работа с программой

Все очень просто!

Работа заключается в задании начальных данных, выполнении решения и
просмотр результатов.

### Задание начальных данных

С помощью клавиш `<+>` `<->` `<*>` `</>` `<:>` можно включать или
выключать операции соответственно сложения, вычитания, умножения,
деления и деления с остатком.

Нажав клавишу `<И>` можно ввести имя. Ввод имени заканчивается нажатием
клавиши `<Enter>`.

Нажав клавишу `<О>` или `<Д>` можно задать верхнюю или нижнюю границу
для результата примеров. Это границы ТОЛЬКО для результатов примеров, -
числа, участвующие в примере могут выходить за эти границы.

Нажатие клавиши `<К>` задается количество примеров.

И, наконец, нажатием клавиши `<C>` задается уровень сложности -
количество секунд, которые даются для вычисления примера.

Все введенные параметры сохраняются в файле `A.DAT` и используются при
следующей загрузке программы.

После задания всех данных для начала вычислений необходимо нажать
клавишу `<Enter>`.

### Выполнение решения

После нажатия клавиши `<Enter>` сразу выдается первый пример. Он
располагается после слова "Пример" и номера выводимого примера. Для
ввода результата необходимо, используя цифровые клавиши, набрать его и
закончить ввод нажатием клавишей `<Enter>`. Это необходимо сделать, пока
в нижней части экрана бежит полоска. Если не успеешь это сделать, на
экране появится большая надпись "ВРЕМЯ ВЫШЛО". При вводе верного ответа
На экране появляется надпись "МОЛОДЕЦ!", а при вводе неверного -
"НЕВЕРНО...". Независимо от результата вычисления напротив слов
"Оценка" выводится текущая оценка по пятибалльной шкале, напротив слов
"Верный ответ:" - правильное значение результата, напротив слов "Верных
ответов:" - количество правильно выполненных заданий. А напротив слова
"Проверка" под буквам "а)" и "б)" - проверка примера с введенным
значением результата. Это позволяет выполнять анализ введенных чисел.

При вводе результата действуют клавиши перемещения курсора вправо и
влево, клавиши `<Home>` и `<End>` для перемещения в начало и в конец
числа, а также клавиша забоя для удаления символа слева от курсора.

Переход к решению следующего примера производится по нажатию клавиши
`<Tab>`. В любой момент времени можно отказаться от решения текущего
примера, нажав клавишу `<Esc>` или от продолжения работы в текущем
сеансе, нажав клавишу `<F10>`.

При нормальном окончании сеанса на экран выводится крупная надпись
"ВАША ОЦЕНКА N", где вместо N ставится оценка.

Для выхода в основное окно программы необходимо нажать клавишу `<Tab>`.

### Просмотр результатов

Результаты каждого сеанса работы сохраняются и записываются в файл. Имя
этого файла образуется по введенному имени, а расширение - числа от `000`
до `999`.

Для просмотра результатов последнего сеанса необходимо нажать клавишу
`<F1>` в основном экране программы. появится окно результатов, в котором
отражена полная информация по решению. В этом окне действуют только
клавиши вертикального перемещения курсора и клавиша `<Esc>` для выхода.
